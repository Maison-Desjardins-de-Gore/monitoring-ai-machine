# AI Machine Monitoring System

## Overview
Monitoring system for AI machine that runs in WSL parent environment (not in Docker containers).

## Repository
- Name: `monitoring-ai-machine`
- URL: `https://github.com/Maison-Desjardins-de-Gore/monitoring-ai-machine`
- Branch: `main`

## Files
- `monitor.sh` - Main monitoring script
- `config.env` - Configuration (bot token)
- `README.md` - Documentation

## Features
- GPU usage monitoring (nvidia-smi)
- VRAM usage monitoring
- Ollama process tracking
- CPU/RAM usage
- Docker container status
- Telegram notifications every 10 minutes

## Telegram Configuration
- Chat ID: `-1003796017691` (already in log channel)
- Bot token: To be configured in `config.env`

## Installation
1. Create GitHub repository
2. Add bot token to `config.env`
3. Install cron job: `*/10 * * * * /path/to/monitor.sh`
4. Test monitoring
